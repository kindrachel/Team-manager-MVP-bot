# Team-manager-MVP-bot
Мощный инструмент для управления командами, организациями и проектами через Telegram. Бот предоставляет полный набор функций для регистрации пользователей, проведения опросов, генерации отчетов, управления челленджами и многое другое. Интегрированная система ИИ помогает в анализе данных и автоматизации процессов.
